🚗 Car Price Analysis Project

🔹 Introduction

This project focuses on analyzing car pricing trends using a dataset of 2500 cars. The objective is to understand how factors such as brand, model, year, fuel type, transmission, condition, and total travel (km driven) influence car prices.

The project follows an end-to-end data analytics workflow using Python for data preprocessing, SQL for analysis, and Power BI for visualization.

🎯 Objectives

Analyze pricing distribution across different car brands
Identify premium and budget segments
Understand the impact of car usage (total travel) on pricing
Perform data cleaning and preprocessing
Extract insights using SQL queries
Build an interactive Power BI dashboard

📊 Dataset Summary

Total Records: 2500 cars

Features include:

Brand

Model

Price

Year

Fuel Type

Transmission

Condition

Total Travel (km driven)

👉 Total Travel represents how many kilometers a car has been driven, which directly impacts resale value.

📈 Key Insights

Average car price is around 52K, indicating a mid-range market
BMW and Tesla are premium brands, while Ford is more affordable
Most expensive cars are close to 100K, showing premium clustering
Prices remain relatively stable across years
Cars with lower total travel (km driven) tend to have higher prices
High mileage cars fall under the affordable segment
Some brands have more cars priced above their average
